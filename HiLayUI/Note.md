## 官网
+ http://www.layui.com
+ 模块化前端UI框架

## 示例
+ 下载, 官网下载压缩包,解压其中的layui文件,全部拷贝到项目中去

+ 引用
```
<head>
    <title>Title</title>
    <link rel="stylesheet" href="layui/css/layui.css">
    <script src="layui/layui.js"></script>
</head>
```

+ 布局
+ 字体图标
+ 按钮
+ 表单
+ 数据表格
+ 导航
+ 动画
